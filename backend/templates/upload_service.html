The following libraries are pre-installed. You do not need to include them in a requirements.txt:

Data Science:

Image/Vision:
• opencv-python
• Pillow / scikit-image
Documents/Misc:
• python-docx / fpdf
• ipython

*Note: If you need a library not listed above, include it in a requirements.txt file within your ZIP upload.
